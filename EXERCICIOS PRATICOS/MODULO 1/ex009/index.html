Trabalhando com listas

listas ordenadas

Com a tag < ol > Voçê cria uma lista ordenada (com númeração) dentro dessa tag Voçê usa a tag < li > (fechar essa tag e opçional) que serão os elementos de dentro da lista Ex: 

            < ol >
            < li > acordar < /li >
            < li > Tomar café < /li >
            < li > estudar < /li >
            < li > trabalhar < /li >
            < /ol >

o resultado vai ser esse:

  1. Acordar
  2. tomar café
  3. estudar
  4. trabalhar
Voçê pode usar alguns parametros para personalizar

Listas não ordenadas

(aquelas que não segue uma numeração ) usando a tag < ul > Voçê cria uma lista não ordenada, dentro dela Voçê tambem usa a tag < li > ex: 

            < ul >
            < li > acordar < /li >
            < li > Tomar café < /li >
            < li > estudar < /li >
            < li > trabalhar </li >
            < /ul >

O resultado será esse:

  • Acordar
  • tomar café
  • estudar
  • trabalhar

Voçê tambem pode personalizar usando seus parametros (parametro type)
